# Android Edge Vision for Logistics

> **Domain:** Logistics

## Overview

On-device mobile solution for inspections using TensorFlow Lite models optimized for mid-range Android phones, supporting container IDs, PPE compliance, and vehicle defect detection.

## Skills & Technologies

- TensorFlow Lite
- Model Quantization
- Object Detection
- Android SDK
- Optical Character Recognition
